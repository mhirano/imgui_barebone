# imgui_test
Use imgui https://github.com/ocornut/imgui with cmake, opencv, opengl 3, sdl2


    git clone https://github.com/lucasw/imgui_test
    cd imgui_test
    git clone https://github.com/ocornut/imgui
    cd ..
    mkdir build_imgui_test
    cd build_imgui_test
    cmake ../imgui_test
    make
    ./imgui_test
